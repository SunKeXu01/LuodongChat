# 数据模型与隐私边界

首版 PostgreSQL 结构位于 `db/migrations/001_initial.sql`。

## 记录内容

- 用户、网关密钥 ID 与密钥哈希；
- 上游及加密后的凭证；
- 用户请求状态、时间、模型、Token 与金额；
- 每次上游尝试的凭证指纹、状态、耗时和成本；
- 请求与尝试之间的一对多关系。

## 禁止记录

- 用户完整提示词；
- 模型输出正文；
- 用户网关密钥明文；
- 上游密钥明文；
- 用户本地 Codex 配置内容。

金额统一存储为整数微单位，避免浮点误差。真实凭证必须由应用层使用 KMS 或独立主密钥加密后写入 `encrypted_secret`。
