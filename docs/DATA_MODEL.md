# 数据模型与隐私边界

首版 PostgreSQL 结构位于 `db/migrations/001_initial.sql`。

## 记录内容

- 用户、网关密钥 ID 与密钥哈希；
- 上游及加密后的凭证；
- 用户请求状态、时间、模型、Token 与金额；
- 每次上游尝试的凭证指纹、状态、耗时和成本；
- 请求与尝试之间的一对多关系。
- 邮箱账号会话、个人资料和余额流水；
- 按账号隔离的跨端会话与聊天消息。

## 禁止记录

- 网关请求日志中的用户提示词和模型输出正文；
- 用户网关密钥明文；
- 上游密钥明文；
- 用户本地 Codex 配置内容。

金额统一存储为整数微单位，避免浮点误差。真实凭证必须由应用层使用 KMS 或独立主密钥加密后写入 `encrypted_secret`。

为实现 Windows 与 Android 跨端同步，`chat_messages` 会保存用户主动在连接器聊天页面中发送和接收的正文；这些内容只能由同一登录账号读取，不会写入网关请求日志。Codex 自身的本地历史不会被上传。
