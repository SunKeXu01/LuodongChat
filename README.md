# ChatGPT 连接器

一款适用于 Windows 的 ChatGPT／Codex 连接工具。无需手动修改配置，通过邮箱验证领取网关密钥后，即可快速开启连接并启动 ChatGPT。

> 当前为预览版本，支持 Windows 10／11 x64。

## 主要功能

- 通过邮箱验证码自助领取网关密钥
- 用户密钥永久有效，不限制每日请求量
- 已有网关密钥可直接填写使用
- 自动检测并安全修改 ChatGPT／Codex 配置
- 修改前自动创建配置备份
- 一键启动已经安装的 ChatGPT
- 未安装 ChatGPT 时提供官方下载入口
- 一键恢复连接前的原始配置
- 启动时自动检查客户端新版本
- 支持在软件内下载、校验并安装更新

## 首次使用

1. 完全退出正在运行的 ChatGPT 或 Codex。
2. 打开 `ChatGPTConnector.exe`。
3. 在“没有密钥？通过邮箱自助领取”区域填写邮箱地址。
4. 点击“获取验证码”，然后查看邮箱中的 6 位验证码。
5. 填写验证码并点击“验证并领取密钥”。
6. 密钥领取成功后，点击“开启连接”。
7. 查看软件显示的配置变更摘要，确认后继续。
8. 显示“已连接，网关验证成功”后，点击“启动 ChatGPT”。

如果已经有网关密钥，可以跳过邮箱验证，直接把密钥填入“网关密钥”输入框，然后点击“开启连接”。

## 关于 ChatGPT 账号

连接器使用 API-Key 模式配置 ChatGPT／Codex，通常无需登录 ChatGPT 账号即可开始使用。

API-Key 模式与普通 ChatGPT 账号模式不同，账号同步、云端聊天记录及部分依赖账号的功能可能不可用。具体界面和可用功能可能随 ChatGPT 客户端版本变化。

## 检查和安装更新

客户端每次启动时都会自动检查新版本：

1. 没有新版本时，按钮显示“检查更新”。
2. 发现新版本时，按钮会变为“立即更新”。
3. 点击“立即更新”并确认后，客户端会自动下载更新。
4. 下载完成后会校验文件完整性，退出旧版本、替换程序并启动新版本。

更新文件只会通过 `https://520skx.com` 下载，并在安装前校验 SHA-256。校验失败时不会替换当前程序。

## 恢复原配置

如果不再使用连接器，或希望回到连接前的状态：

1. 完全退出 ChatGPT 或 Codex。
2. 打开连接器。
3. 点击“恢复原配置”。
4. 确认最近一次备份后等待恢复完成。
5. 重新启动 ChatGPT 或 Codex。

如果连接器检测到相关配置已经被其他程序修改，会停止自动恢复并提示发生冲突，避免覆盖用户的新配置。

## 常见问题

### 收不到邮箱验证码

- 检查邮箱地址是否填写正确。
- 查看垃圾邮件或广告邮件目录。
- 等待片刻后重新获取验证码。
- 验证码仅用于本人领取密钥，请勿转发给他人。

### 点击“开启连接”后提示 ChatGPT 正在运行

需要从系统托盘完全退出 ChatGPT／Codex，而不只是关闭窗口。退出后重新点击“开启连接”。

### 无法启动 ChatGPT

点击“下载 ChatGPT”前往官方下载页面并完成安装。安装完成后重新打开连接器，再点击“启动 ChatGPT”。

### 连接失败

请检查网络是否正常，并确认网关密钥没有多余空格。如果仍然失败，可以稍后重试或重新通过邮箱领取密钥。

### 密钥应该如何保管

网关密钥相当于个人访问凭证，请勿截图、公开或发送给他人。连接器不会要求用户把密钥提交到 GitHub。
